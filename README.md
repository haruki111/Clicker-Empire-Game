# Clicker Empire Game
## Description
[Recursion](https://recursionist.io/)の課題で作成した、仕事や投資、不動産等でお金を稼ぐシミュレーションゲームです。  
20歳のユーザーの夢は一攫千金を獲得し、帝国を築くことです。  
そのために、ファーストフード店でハンバーガーを焼きながら、投資、不動産の購入をします。   
# Features
追加機能  
・ゲームスピードの変更　(1,2,5,10,50,100倍)  
・購入時に最大購入数をクリックで指定  
・帝国を購入し、Game Clear
